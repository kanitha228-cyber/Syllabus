**Pre-requisite:**

Digital Logic Design, Computer Organization and Architecture, Microprocessor, Programming in C.

**Overview of Course:**

This course introduces the fundamental concepts of Embedded Systems, including embedded hardware architecture, ARM-based microcontrollers, embedded software development, communication interfaces, peripheral interfacing, and real-time operating systems. Students will understand the design and development of embedded applications using Embedded C, analyze wired and wireless communication protocols, interface sensors and peripheral devices, and explore modern embedded applications in industrial automation, healthcare, automotive electronics, consumer electronics, and Internet of Things (IoT).

**Course Objectives:**

**CO1:** Understand the architecture and organization of embedded systems, CISC and RISC based microcontrollers, and embedded hardware components.

**CO2:** Explain ARM architecture, embedded software development tools, and firmware development process for embedded applications.

**CO3:** Analyze wired and wireless communication protocols used in embedded systems.

**CO4:** Understand Real-Time Operating Systems (RTOS), task scheduling, synchronization mechanisms and interrupt handling.

**CO5:** Develop embedded applications using Embedded C and interface sensors, actuators, and peripheral devices.

**CO6:** Analyze modern embedded system design methodologies and emerging embedded technologies for industrial and IoT applications.

**Syllabus:**

| **Unit – 1** |
| **Embedded Hardware Architecture:** Introduction to Embedded Systems, Characteristics and Classification of Embedded Systems, Embedded Design Methodology. **CISC Architecture:** Introduction to MCS-51 Family, 8051 Microcontroller Architecture, Memory Organization, I/O Ports, Timers/Counters, Interrupts, Serial Data Communication. **RISC Architecture:** Introduction to ARM Architecture, ARM Cortex-M Processor Family, Memory Organization, Timers, Interrupts, GPIO. Comparison of CISC and RISC Architectures. **Self Learning:** AVR and ESP32 Microcontrollers. |
| **Unit – 2** |
| **ARM Architecture and Embedded Software Development:** LPC4088 Architecture, Memory Organization, GPIO, Timers, Interrupt Controller, Embedded Software Development Process, Embedded C Programming Fundamentals, Cross Compiler, Linker, Loader, Debugger. **Software Development Tools:** IDE Tools, ISP Tools, Flash Programming Tools. **Self Learning:** CMSIS Framework and Keil µVision IDE. |
| **Unit – 3** |
| **Communication Interfaces and Peripheral Interfacing:** **Wired Communication Protocols:** RS-232, UART, I²C, SPI, USB(Introduction). **Wireless Communication Protocols:** Bluetooth Low Energy (BLE), IEEE 802.11 (Wi-Fi), LoRaWAN(Overview). **Peripheral Interfacing:** Digital Input and Output Devices, ADC Interface, DAC Interface, Sensor Interfacing, Embedded C for Peripheral Control. **Self Learning:** MQTT Protocol and IoT Communication. |
| **Unit – 4** |
| **Real-Time Operating Systems and Embedded Applications:** Operating System Basics, Kernel and its Subsystems, Kernel Space and User Space, Types of RTOS, Functions of RTOS, Task, Process and Thread Management, Interrupt Handling, Multitasking and Task Scheduling, Semaphores, Mutexes and Message Queues, Overview of FreeRTOS and Embedded Linux. Applications of Embedded Systems in Consumer Electronics, Automotive Systems, Healthcare, Industrial Automation and Internet of Things (IoT). **Self Learning:** TinyML, Edge AI and RISC-V Embedded Systems. |
